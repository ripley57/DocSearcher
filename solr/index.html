Solr Administration

Solr Luke UI http://localhost:8983/solr/admin/luke
Note: To enable the Luke UI, add the following to the core's solrconfig.xml file (e.g. example/solr/collection1/conf/solrconfig.xml):

<requestHandler name="/admin/luke" class="org.apache.solr.handler.admin.LukeRequestHandler" />

See https://wiki.apache.org/solr/LukeRequestHandler
